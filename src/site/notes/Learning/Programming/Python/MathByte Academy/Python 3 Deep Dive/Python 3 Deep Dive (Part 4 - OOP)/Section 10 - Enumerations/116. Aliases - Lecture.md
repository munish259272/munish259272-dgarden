# Enumeration Aliases in Python

## Understanding Aliases

Enums support **aliases**—multiple names for the same member:

```python
from enum import Enum

class Color(Enum):
    RED = 1
    CRIMSON = 1
    CARMINE = 1
    BLUE = 2
    AQUAMARINE = 2
```

- **Oddity**: `RED`, `CRIMSON`, and `CARMINE` all have value `1`—aren’t members supposed to be unique?
- **Answer**: **Only master members are unique**; others are aliases.
- **Key Point**: **Aliases point to the first defined member**.
	- Color.CRIMSON -> Color.RED 
	- Color.CARMINE -> Color.RED 
	- Color.AQUAMARINE -> Color.BLUE

---

## Master Members vs. Aliases

Only **master members** (first defined) count:

```python
print(list(Color))  # [<Color.RED: 1>, <Color.BLUE: 2>]
print(Color.CRIMSON is Color.RED)  # True
print(Color.CARMINE is Color.CRIMSON)  # True
print(Color.__members__) # {'RED': <Color.RED: 1>, 'CRIMSON': <Color.RED: 1>, 'CARMINE': <Color.RED: 1>, 'BLUE': <Color.BLUE: 2>, 'AQUAMARINE': <Color.BLUE: 2>}
```

- **Count**: Just `RED` and `BLUE`—aliases (`CRIMSON`, etc.) are excluded from iteration.
- **Identity**: Aliases reference the master (`Color.CRIMSON` is `Color.RED`).
- **Key Point**: **Aliases share the master’s identity**.

---

## Lookups with Aliases

Lookups return the **master member**:

```python
print(Color(1))          # Color.RED
print(Color["CRIMSON"])  # Color.RED
print(Color.CARMINE in Color)  # True
```

- **By Value**: `Color(1)` → `RED`, not `CRIMSON` or `CARMINE`.
- **By Name**: `Color["CRIMSON"]` → `RED`.
- **Containment**: Aliases test `True` for membership.
- **Key Point**: **Lookups resolve to the master**.

---

## Inspecting Aliases

Use `__members__` to see all names:

```python
print(Color.__members__)
# {'RED': <Color.RED: 1>, 'CRIMSON': <Color.RED: 1>, 'CARMINE': <Color.RED: 1>, ...}
print(Color.RED.name) # 'RED'
```

- **Details**: Keys are all names; values point to masters (`RED`, `BLUE`).
- **Key Point**: **`__members__` reveals alias mappings**.

---

## Ensuring Unique Values

Prevent aliases with the `@unique` decorator:

```python
from enum import unique

@unique
class UniqueColor(Enum):
    RED = 1
    CRIMSON = 1  # ValueError: duplicate values found
```

- **Effect**: Raises `ValueError` at class creation(At the time module is loaded or the class is compiled) if values repeat.
- **Key Point**: **`@unique` enforces distinct values**.
