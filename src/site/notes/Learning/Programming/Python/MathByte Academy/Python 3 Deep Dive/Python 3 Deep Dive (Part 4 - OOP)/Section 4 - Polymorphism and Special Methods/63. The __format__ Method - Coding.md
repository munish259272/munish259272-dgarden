# Formatting Objects with `__format__`

## Using `format()` with Built-ins

The `format()` function shapes how values appear:

```python
# Floats
print(format(3.14159, ".2f"))  # 3.14 (2 decimals)
print(format(3.14159, ".20f")) # 3.14159000000000000000 (20 decimals)

# Datetime
from datetime import datetime
now = datetime.utcnow()
print(format(now, "%A %Y-%m-%d %I:%M %p"))  # e.g., Sunday 2025-03-28 02:45 PM
```

- **Spec**: Controls precision (`.2f`) or pattern (`%A %Y-%m-%d`).
- **Key Point**: ** `format()` tailors output precisely**.

---

## Adding `__format__` to Your Class

Support `format()` in custom classes with `__format__`:

```python
from datetime import date

class Person:
    def __init__(self, name, dob):
        self.name = name
        self.dob = dob  # Date object
    
    def __repr__(self):
        print("repr called")
        return f"Person(name={self.name}, dob={self.dob.isoformat()})"
    
    def __str__(self):
        print("str called")
        return f"Person {self.name}"
    
    def __format__(self, date_spec):
        print(f"format called with spec={date_spec}")
        dob_str = format(self.dob, date_spec)  # Delegate to date
        return f"name={self.name}, dob={dob_str}"

p = Person("Alex", date(1910, 10, 20))
```

- **`__format__(self, spec)`**: Takes a format spec (e.g., `%B %d, %Y`).
- **Delegation**: Passes `spec` to `self.dob`’s `format()`.

---

## Testing It Out

Try different calls:

```python
print(str(p))          # str called, Person Alex
print(repr(p))         # repr called, Person(name=Alex, dob=1910-10-20)
print(format(p, "%B %d, %Y"))  # format called with spec=%B %d, %Y, name=Alex, dob=October 20, 1910
print(format(p))       # format called with spec=, name=Alex, dob=1910-10-20
```

- **Output**:
  - `str()` → Simple name.
  - `repr()` → Detailed with ISO date.
  - `format(spec)` → Custom date format.
  - `format()` → Default (empty spec).

---

## Empty Spec Behavior

No spec means `""`, falling back to `str()` or similar:

- `format(p)` → `date_spec` is `""`.
- `format(self.dob, "")` → Uses `date`’s default (ISO-like string).
- **Chain**: `__format__` → delegated `format()` → `str()` if no spec handling.
- **Key Point**: **Empty spec defaults sensibly**.

---

## Why Delegate?

Parsing custom specs (e.g., `"short"`, `"long"`) is complex:
- Requires writing a mini-language—beyond basic needs.
- **Solution**: Delegate to types like `float` or `date` that already handle specs.
- **Takeaway**: **Delegation keeps it simple**.
