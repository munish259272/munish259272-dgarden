# Prerequisites for This Course

## **Who is This Course For?**

This course is **not for beginners**. If you’ve only completed a **boot camp** or **introductory courses** and **lack practical experience** in Python, this course **is not the best fit** for you. Instead, gain **hands-on experience** first, then return to this deep dive.

## **Required Knowledge**

To benefit from this course, you should already have a solid understanding of the following **computer science concepts**:

- **Programming Paradigms**: Functional programming, object-oriented programming (OOP)
- **Data Structures**: Lists, dictionaries, sets, tuples, stacks, queues, and trees
- **Basic CS Fundamentals**: Algorithms, memory management, namespaces, and scopes

Though these may not be explicitly covered in the course, they form the foundation for deeper Python understanding.

## **Practical Python Experience**

This is **not a cookbook-style course** that teaches how to solve specific problems. Instead, it focuses on **understanding Python’s mechanics** and **idiomatic programming**. You should have experience working on **real-world Python projects**, not just small exercises.

## **Key Python Concepts You Should Know**

### **Fundamental Python Knowledge**

You should be comfortable with the following topics:

- **Scopes and Namespaces**
- **Boolean Truth Values** (associated with every Python object)
- **Memory Management**
    - `id()` function and object identity
    - [[Learning/Programming/Python/MathByte Academy/Python 3 Deep Dive/Python 3 Deep Dive (Part 1 - Functional)/Section 3 - Variables and Memory/17. Reference Counting\|Reference counting]], [[Learning/Programming/Python/MathByte Academy/Python 3 Deep Dive/Python 3 Deep Dive (Part 1 - Functional)/Section 3 - Variables and Memory/23. Shared References and Mutability\|shared references]], and pointers
- **Equality Operators**: [[Learning/Programming/Python/MathByte Academy/Python 3 Deep Dive/Python 3 Deep Dive (Part 1 - Functional)/Section 3 - Variables and Memory/24. Variable Equality\|== vs. is]]

### **Functional Programming Concepts**

- **Function Arguments**:
    - Positional, keyword-only arguments
    - Default vs. non-default arguments
- **Lambdas** (anonymous functions)
- **Packing and Unpacking** of iterables
    
    ```python
    a, *b, c = [1, 2, 3, 4]  # Extended unpacking
    ```
    
- **Closures & Nested Scopes**
- **Free Variables** and **Decorators**
    - Creating and using decorators
    - **Decorator factories & classes**
- **Common Built-in Functions**
    - `zip()`, `sorted()`, `any()`, `all()`
    - `itertools` module utilities

### **Object-Oriented Programming (OOP) Essentials**

- **Classes and Instances**
- **Special Methods (Dunder Methods)**
    - `__eq__`, `__hash__`, `__lt__`, etc.
- **Comprehensions**
    - List, dictionary comprehensions
    - Understanding comprehensions as functions
- **Generators & Iterators**
    - `yield`, lazy iteration
    - `__iter__()` and `__next__()`
- **Context Managers**
    - `with` statement, implementing `__enter__()` and `__exit__()`
- **Mapping Types**
    - Dictionaries, sets, hashability
    - `collections.defaultdict`
- **Object Equality & Hashing**

### **Working with Modules & Environments**

- **Importing Modules and Symbols**
- **Virtual Environments**
    - Setting up and using `venv`
    - Installing packages with `pip`

## **Development Tools**

- **Jupyter Notebooks**
    - Many course examples use Jupyter, which is **free** to install and use.
- **Git & GitHub**
    - Course materials are available on GitHub.
    - You should be comfortable using Git (`git clone`, `git pull`).

## **Final Notes**

This course assumes a **strong foundation in Python**. If these concepts are new to you, consider **revisiting Python fundamentals** before proceeding.
