## **Course Overview**

### **What is this Course About?**

This course is designed to help you become an **expert Python developer** by focusing on:

- The **Python language**
- The **standard library**
- The **canonical CPython implementation** (version **3.6 or above**)
- Writing **idiomatic Python**
- Gaining **deep insights** into the language and its mechanics

### **Who is this Course For?**

This is **not** an introductory course. There are **prerequisites** you should be familiar with before starting:

- Prior **practical Python programming** experience
- Ability to **solve real-world problems**
- Understanding of **what you want to learn** to enhance your **professional career**

⚠️ **Note:** This is **not a cookbook-style course** that teaches programming from scratch. Instead, it focuses on in-depth **concepts and techniques**.

---

## **Topics Covered**

### **1️⃣ Object-Oriented Programming (OOP) in Python**

- **Classes & Objects**
- **Creating instances** of classes
- **Initialization & object lifecycle**
- **Functions, methods, properties, and attributes**
- **Data attributes vs Callable attributes** (important distinction)

### **2️⃣ Polymorphism & Special Dunder Methods**

- How Python enables polymorphism
- Understanding and implementing **Dunder (__) methods**
- Defining **custom operators** (e.g., `+`, `-`, `*`) for user-defined classes

### **3️⃣ Single & Multiple Inheritance**

- **Single inheritance** (primary focus)
- Understanding why **multiple inheritance** is often unnecessary
- Comparison with languages like **Java**, which do not support multiple inheritance

### **4️⃣ Descriptors**

- **Fundamentals of descriptors** in Python
- How they enable **properties and methods**
- Difference between **data descriptors & non-data descriptors**
- Why understanding descriptors is crucial for **advanced OOP**

### **5️⃣ Enumerations (Enums)**

- Managing **collections of related constants**
- Using **global module-level attributes** vs **enumerations**
- Advantages of **extending enumerations** for custom use cases

### **6️⃣ Exception Handling**

- **Raising & catching exceptions** using `try-except` blocks
- Understanding **exception hierarchies**
- Creating **custom exceptions** tailored to specific problems
