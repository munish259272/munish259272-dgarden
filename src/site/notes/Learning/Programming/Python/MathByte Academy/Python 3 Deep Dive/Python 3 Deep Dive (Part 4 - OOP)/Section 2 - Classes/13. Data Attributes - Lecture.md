# Data Attributes: Classes vs. Instances

## Defining a Class with Data

Here’s a simple class with a data attribute:
```python
class MyClass:
    language = "Python"
```

- **`language`**: A **class attribute**, stored in the class’s namespace.
- **Namespace**: `MyClass.__dict__` contains `{"language": "Python", ...}`.

**Key Point**: **Class attributes** belong to the class itself.

---

## Creating an Instance

Instantiate the class:
```python
my_object = MyClass()
```

- **Callable**: `MyClass()` returns an instance, thanks to Python’s built-in behavior.
- **Instance Namespace**: `my_object.__dict__` starts empty (`{}`).

**Takeaway**: **Instances have their own namespace**, separate from the class.

---

## Accessing Attributes

### Class Attributes
Access via the class:
```python
print(MyClass.language)  # "Python"
```

- Python checks `MyClass.__dict__`, finds `"language"`, and returns its value.

### Instance Access
Access via the instance:
```python
print(my_object.language)  # "Python"
```

- **Lookup Process**:
  1. Check `my_object.__dict__`—it’s empty.
  2. Look up to `MyClass.__dict__`—finds `"language": "Python"`.
  3. Returns `"Python"`.

**Key Point**: Instances **fall back to class attributes** if not overridden.

---

## Setting Instance Attributes

Set an attribute on the instance:
```python
my_object.language = "Java"
print(my_object.__dict__)  # {'language': 'Java'}
```

- **Instance Attribute**: Now `my_object` has its own `"language": "Java"`.
- **Class Unchanged**: `MyClass.language` remains `"Python"`.

Check both:
```python
print(my_object.language)  # "Java" (instance attribute)
print(MyClass.language)   # "Python" (class attribute)
```

**Takeaway**: **Instance attributes override class attributes** for that instance.

---

## Lookup Order

Python’s attribute lookup:
1. **Instance Namespace**: `my_object.__dict__`—returns value if found.
2. **Class Namespace**: `MyClass.__dict__`—falls back if instance lacks it.

Example with a new instance:
```python
other_object = MyClass()
print(other_object.__dict__)  # {}
print(other_object.language)  # "Python" (from class)
```

- **Empty**: `other_object` has no `"language"`.
- **Fallback**: Gets `"Python"` from `MyClass`.

**Key Point**: **Each instance has a unique namespace**.
