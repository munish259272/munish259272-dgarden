# Customizing Formatting with `__format__`

## Python’s `format()` Basics

The built-in `format()` function lets you **control how objects are displayed**:

```python
num = 10.555
print(format(num, ".2f"))  # 10.56 (2 decimal places)
print(format(42, "d"))     # 42 (decimal integer)
from datetime import datetime
print(format(datetime(2023, 1, 1), "%Y-%m-%d"))  # 2023-01-01
```

- **Spec**: The second argument (e.g., `.2f`, `d`, `%Y-%m-%d`) defines the format.
- **Types**: Works with floats, integers, dates, etc.
- **Resource**: Check Python’s [format specification docs](https://docs.python.org/3/library/string.html#formatspec) for details.

---

## Adding `__format__` to Classes

Make your objects work with `format()` by implementing `__format__`:

```python
class Temperature:
    def __init__(self, celsius):
        self.celsius = celsius
    
    def __format__(self, spec):
        if spec == "":
            return str(self.celsius)  # Default if no spec
        elif spec == ".1f":
            return f"{self.celsius:.1f}°C"
        return str(self.celsius)  # Fallback

temp = Temperature(23.75)
print(format(temp, ".1f"))  # 23.8°C
print(format(temp))        # 23.75
```

- **`__format__(self, spec)`**: Called by `format(temp, spec)`.
- **Spec Handling**: `spec` is the format string (e.g., `.1f`). If empty, it defaults to `""`.
- **Key Point**: **You decide how `spec` shapes the output**.

---

## Default Behavior

No spec? It falls back to `str()`:

```python
class Simple:
    def __str__(self):
        return "Simple Object"

obj = Simple()
print(format(obj))  # Simple Object
```

- **Chain**: `format(obj, "")` → `__format__` (if defined) → `str(obj)` → `__str__` (or `__repr__` if no `__str__`).
- **Takeaway**: **Empty spec leans on string conversion**.

---

## Delegating Formatting

Instead of writing complex specs, **delegate to existing types**:

```python
class Distance:
    def __init__(self, meters):
        self.meters = meters
    
    def __format__(self, spec):
        return format(self.meters, spec)  # Delegate to float

dist = Distance(123.456)
print(format(dist, ".2f"))  # 123.46
print(format(dist))        # 123.456
```

- **Why**: `float` already handles specs like `.2f`. Reuse it!
- **Common**: Simplest way to support formatting without reinventing the wheel.
- **Key Point**: **Leverage built-in formatting**.

---

## Complexity Note

Writing your own format spec parser (e.g., handling custom patterns like `temp:precision`) is tricky and rare—check the docs if curious, but delegation is usually enough.
