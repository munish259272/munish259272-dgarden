# Applying Properties in Python

## Read-Only Properties

A **read-only property** has a getter but no setter, preventing direct modification:

```python
class Example:
    def __init__(self, language):
        self._language = language
    
    @property
    def language(self):
        return self._language

e = Example("Python")
print(e.language)  # Python
# e.language = "Java"  # AttributeError: can't set attribute
```

- **`_language`**: A backing variable with an underscore to signal “don’t touch.”
- **Caveat**: Python doesn’t enforce privacy—`e._language = "Java"` still works, but it’s discouraged.

**Key Point**: **Read-only properties rely on convention**, not strict enforcement.

## Computed Properties

Properties can compute values on demand, like a circle’s area:

```python
from math import pi

class Circle:
    def __init__(self, radius):
        self.r = radius
    
    @property
    def area(self):
        return pi * self.r ** 2

c = Circle(1)
print(c.area)  # 3.14159...
```

- **No `()`**: `c.area` feels like an attribute, not a method (`c.area()`).
- **Dynamic**: Recalculates each time `c.area` is accessed.

**Takeaway**: **Computed properties enhance readability** for derived values.

## Caching Computed Properties

To avoid recomputing, cache the result and control updates with a setter:

```python
from math import pi

class Circle:
    def __init__(self, radius):
        self._r = radius
        self._area = None  # Cache starts empty
    
    @property
    def radius(self):
        return self._r
    
    @radius.setter
    def radius(self, r):
        if r < 0:
            raise ValueError("Radius cannot be negative")
        self._r = r
        self._area = None  # Invalidate cache
    
    @property
    def area(self):
        if self._area is None:  # Lazy computation
            self._area = pi * self._r ** 2
        return self._area

c = Circle(1)
print(c.area)  # Computes: 3.14159...
print(c.area)  # Cached: 3.14159...
c.radius = 2  # Resets cache
print(c.area)  # Recomputes: 12.566...
```

- **`_area`**: Stores the computed value, initially `None`.
- **Lazy Evaluation**: Calculates only when needed, then reuses the cache.
- **Setter Role**: Resets `_area` when `radius` changes.

**Key Point**: **Setters manage cache invalidation** for efficiency.
