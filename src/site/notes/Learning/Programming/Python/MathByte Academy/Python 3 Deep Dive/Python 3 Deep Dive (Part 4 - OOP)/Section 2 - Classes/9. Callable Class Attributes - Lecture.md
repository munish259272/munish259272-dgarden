# Callable Class Attributes in Python

## Adding Callable Attributes

So far, we’ve seen **data attributes** like strings or numbers. Now, let’s explore **callable attributes**—attributes that are functions.

Define a class with a function as an attribute:
```python
class MyClass:
    language = "Python"
    def say_hello():
        return "Hello World"
```

- **`language`**: A data attribute with value `"Python"`.
- **`say_hello`**: A **callable attribute**—a function tied to the class.

**Key Point**: **Attributes can be any object**, including functions.

---

## Inspecting the Class

Check the class’s namespace:
```python
print(MyClass.__dict__)
# Output: mappingproxy({'language': 'Python', 'say_hello': <function MyClass.say_hello ...>, ...})
```

- **`say_hello`**: Appears as a function in `__dict__`, confirming it’s an attribute of the class.

**Takeaway**: **Callable attributes** are stored like any other attribute.

---

## Calling the Function

Since `say_hello` is a class attribute, we access and call it via the class object (not instances). Here’s how:

### 1. Via `__dict__`
```python
func = MyClass.__dict__["say_hello"]
print(func())  # "Hello World"
```

- Or in one line:
```python
print(MyClass.__dict__["say_hello"]())  # "Hello World"
```

- ⚠️ **Warning**: Not recommended—`__dict__` is for inspection, not routine access.

### 2. Via `getattr()`
```python
func = getattr(MyClass, "say_hello")
print(func())  # "Hello World"
```

### 3. Via Dot Notation
Simplest and most common:
```python
print(MyClass.say_hello())  # "Hello World"
```

**Key Point**: **Dot notation** is the cleanest way to call class functions.

---

## Class Focus

All examples here work on the **class object** (`MyClass`), not instances. We’re manipulating the class itself.

**Takeaway**: **Callable attributes** belong to the class, not its instances.
