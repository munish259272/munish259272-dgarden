Let’s dive into **extending functionality** through inheritance in Python—a quick and simple concept.

Inheritance lets us create a **specialized class** that builds on a more general one. Start with a basic `Person` class, which inherits from `object` by default:
```python
class Person:
    pass
```

Now, create a `Student` class that inherits from `Person` and adds new functionality:
```python
class Student(Person):
    def study(self):
        return "Studying hard!"
```
Here, `Student` **extends** `Person` by adding a `study` method. While `Person` has only what `object` provides, `Student` gains this extra capability, making it more specialized. All `Student` instances can use `study`, but `Person` instances cannot.

This extension isn’t limited to methods—you can add **class attributes** too—but methods like `study` are where it shines.

Let’s test it:
```python
p = Person()
s = Student()
# print(p.study())  # This would fail—Person has no study method
print(s.study())    # Outputs: "Studying hard!"
```

That’s it—**extending** is just about adding new features to a subclass, and this quick demo shows how straightforward it is!
