# Enumerations in Python

## Why Enumerations?

Instead of scattering constants like this:

```python
RED = 1
GREEN = 2
BLUE = 3
COLORS = [RED, GREEN, BLUE]
```

Enumerations **group related constants** with added benefits.

- **Question**: Why use enumerations over plain variables?
- **Answer**: They offer **structure, safety, and extra features**.
