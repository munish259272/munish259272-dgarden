# Diving into Python Descriptors

## What Are Descriptors?

Descriptors are **key to Python’s internals**, powering features like:

- **Properties**
- **Methods**
- **Slots**
- **Functions**

We’ll explore them in this section to understand how they work under the hood.

---

## Types of Descriptors

There are two main kinds:

1. **Non-Data Descriptors**: Define only `__get__`—think method lookups.
2. **Data Descriptors**: Define `__get__` and `__set__` (or `__delete__`)—like properties or slots.

- **Difference**: Their behavior varies in attribute resolution (more on that soon).
