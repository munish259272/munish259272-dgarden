# Working with Enum Aliases

## Creating Aliases

Define an enum with aliases using repeated values:

```python
from enum import Enum

class NumSides(Enum):
    TRIANGLE = 3
    RECTANGLE = 4
    SQUARE = 4
    RHOMBUS = 4
```

- **Result**: `RECTANGLE` is the master; `SQUARE` and `RHOMBUS` are aliases.
- **Key Point**: **Aliases share the first member’s identity**.

---

## Checking Aliases

Aliases are the same object as the master:

```python
print(NumSides.RECTANGLE is NumSides.SQUARE)   # True
print(NumSides.RHOMBUS is NumSides.SQUARE)     # True
print(NumSides.SQUARE in NumSides)             # True
```

- **Identity**: `SQUARE` and `RHOMBUS` are `RECTANGLE`.
- **Containment**: Aliases test `True` for membership.
- **Key Point**: **Aliases behave like their master**.

---

## Lookup Behavior

Lookups return the **master member**:

```python
print(NumSides(4))          # NumSides.RECTANGLE
print(NumSides["SQUARE"])   # NumSides.RECTANGLE
```

- **By Value**: `NumSides(4)` → `RECTANGLE`, not `SQUARE`.
- **By Name**: `NumSides["SQUARE"]` → `RECTANGLE`.
- **Key Point**: **Master member wins in lookups**.

---

## Inspecting Members

Use `__members__` to see all names:

```python
print(NumSides.__members__)
# {'TRIANGLE': <NumSides.TRIANGLE: 3>, 'RECTANGLE': <NumSides.RECTANGLE: 4>,
#  'SQUARE': <NumSides.RECTANGLE: 4>, 'RHOMBUS': <NumSides.RECTANGLE: 4>}
print(NumSides.__members__["SQUARE"])  # NumSides.RECTANGLE
```

- **Details**: Shows all keys, but aliases point to `RECTANGLE`.
- **Iteration**: `list(NumSides)` gives only `TRIANGLE` and `RECTANGLE`.
- **Key Point**: **`__members__` exposes aliases; iteration skips them**.

---

## Practical Alias Use

Map system statuses with aliases:

```python
class Status(Enum):
    READY = 1
    RUNNING = 2
    BUSY = 2
    PROCESSING = 2
    OKAY = 3
    FINISHED = 3
    RAN_OKAY = 3
    ERRORS = 4
    FINISHED_WITH_ERRORS = 4
    ERRORED = 4

print(list(Status))          # [READY, RUNNING, OKAY, ERRORS]
print(Status["BUSY"])        # Status.RUNNING
print(Status["PROCESSING"])  # Status.RUNNING
print(Status["RAN_OKAY"])    # Status.OKAY
```

- **Use Case**: Normalize API strings (`busy`, `processing`) to app statuses (`RUNNING`).
- **Flexibility**: Values can be ints, strings—**doesn’t matter**.
- **Key Point**: **Aliases unify external data**.

---

## Ensuring No Aliases

Use `@unique` to enforce distinct values:

```python
from enum import unique

@unique
class UniqueStatus(Enum):
    READY = 1
    RUNNING = 1  # ValueError: duplicate values found
```

- **Effect**: Raises `ValueError` at compile time if aliases exist.
- **Key Point**: **`@unique` adds a safety check**.
