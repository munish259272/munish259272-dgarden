# Creating a Database-Powered API Endpoint

Now that our database and table have been created and populated with records, let's create our first FastAPI endpoint that reads data directly from the database. This endpoint will return all todos stored in `todos.db`.

---

# Understanding `create_all()`

Earlier, we created the database using:

```python
models.Base.metadata.create_all(bind=engine)
```

This command:

- Checks whether the database and tables already exist.
    
- Creates them if they do not exist.
    

However:

⚠️ It **does not automatically modify existing tables**.

For example, if you later change your model:

```python
class Todos(Base):
    __tablename__ = "todos"

    id = Column(Integer, primary_key=True)
    title = Column(String)

    # Newly added column
    due_date = Column(String)
```

`create_all()` will **not update** the existing table structure.

---

> [!warning]  
> `create_all()` is useful during development, but it is **not a database migration tool**.
> 
> If your table structure changes:
> 
> - Delete and recreate the database (development only)
>     
> - Or use migration tools such as **Alembic** (production environments)
>     

---

# Creating a Database Dependency

Before an API endpoint can query the database, it needs a database connection.

Create a dependency:

```python
from database import SessionLocal

def get_db():
    db = SessionLocal()

    try:
        yield db
    finally:
        db.close()
```

---

# Understanding `yield`

This is one of the most important FastAPI patterns.

```python
def get_db():
    db = SessionLocal()

    try:
        yield db
    finally:
        db.close()
```

Execution flow:

```text
Create DB Session
        │
        ▼
yield db
        │
        ▼
API Endpoint Runs
        │
        ▼
Response Sent
        │
        ▼
db.close()
```

---

## Why Use `yield`?

FastAPI pauses execution at:

```python
yield db
```

and gives the database session to the endpoint.

After the endpoint finishes and the response is sent:

```python
db.close()
```

runs automatically.

---

> [!tip]  
> Think of `yield` as:
> 
> ```text
> "Here is the database connection.
> Use it.
> When you're done,
> I'll automatically clean everything up."
> ```

---

# Why Close Database Connections?

Database connections are limited resources.

Bad practice:

```python
db = SessionLocal()

# Never closed
```

If thousands of requests arrive:

```text
Request 1 → Open Connection
Request 2 → Open Connection
Request 3 → Open Connection
...
```

Eventually:

```text
Too Many Connections
```

---

> [!success]  
> Using:
> 
> ```python
> try:
>     yield db
> finally:
>     db.close()
> ```
> 
> ensures every connection is cleaned up properly.

# Dependency Injection

FastAPI provides a feature called **Dependency Injection**.

Dependency Injection means:

> "Run some code first and provide the result to my endpoint."

Example:

```python
Depends(get_db)
```

means:

```text
Run get_db()
       ↓
Create DB Session
       ↓
Pass Session To Endpoint
```

---

# Creating the Endpoint

```python
@app.get("/")
async def read_all(
    db: Annotated[
        Session,
        Depends(get_db)
    ]
):
    return db.query(Todos).all()
```

---

# Breaking Down the Endpoint

## Step 1: Open Database Session

```python
Depends(get_db)
```

calls:

```python
get_db()
```

which:

```python
db = SessionLocal()
```

creates a database connection.

---

## Step 2: Inject Session Into Endpoint

FastAPI automatically provides:

```python
db
```

to the endpoint.

---

## Step 3: Query Database

```python
db.query(Todos)
```

means:

> Query the `Todos` table.

---

## Step 4: Return All Records

```python
.all()
```

means:

> Return every row.

Equivalent SQL:

```sql
SELECT * FROM todos;
```

# Avoiding Repeated Dependency Code

This annotation is long:

```python
db: Annotated[
    Session,
    Depends(get_db)
]
```

Since nearly every endpoint needs database access, we can store it in a variable.

---

# Create a Reusable Dependency

```python
DBDependency = Annotated[
    Session,
    Depends(get_db)
]
```

---

# Simplified Endpoint

```python
@app.get("/")
async def read_all(
    db: DBDependency
):
    return db.query(Todos).all()
```

Much cleaner.

# SQL Equivalent

This SQLAlchemy code:

```python

```

is roughly equivalent to:

```sql
SELECT *
FROM todos;
```

SQLAlchemy translates Python code into SQL behind the scenes.
