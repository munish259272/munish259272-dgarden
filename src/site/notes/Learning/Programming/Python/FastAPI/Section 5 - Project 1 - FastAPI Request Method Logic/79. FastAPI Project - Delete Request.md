# 🗑️ Implementing DELETE Requests in FastAPI

At this point, Project 1 has covered all major CRUD operations:

|HTTP Method|Purpose|
|---|---|
|**GET**|Read data|
|**POST**|Create data|
|**PUT**|Update data|
|**DELETE**|Remove data|

So far we have:

✅ Used **path parameters** and **query parameters** with GET requests.

✅ Created new books using POST requests.

✅ Updated existing books using PUT requests.

✅ Now we will implement the final CRUD operation: **DELETE**.

---

# 🎯 Goal

We want to delete a book from our books list by providing its title in the URL.

Example:

```http
DELETE /books/delete_book/Title Four
```

Here:

```text
Title Four
```

is a **path parameter** that identifies which book should be deleted.

---

# 🏗️ Creating the DELETE Endpoint

```python
@app.delete("/books/delete_book/{book_title}")
async def delete_book(book_title: str):
```

### What's Happening Here?

- `@app.delete(...)` creates a DELETE endpoint.
- `{book_title}` is a **dynamic path parameter**.
- FastAPI automatically extracts the value from the URL.
- The extracted value is passed into the function as `book_title`.

For example:

```http
/books/delete_book/Title Four
```

becomes:

```python
book_title = "Title Four"
```

inside the function.

---

# 🔍 Searching for the Book

To delete a book, we first need to find it.

```python
for i in range(len(books)):
```

This loops through every book in the list.

---

# 🎯 Finding a Matching Title

```python
if books[i].get("title").casefold() == book_title.casefold():
```

### Why use `casefold()`?

`casefold()` performs a **case-insensitive comparison**.

Example:

```python
"Title Four".casefold()
```

and

```python
"title four".casefold()
```

both become:

```python
"title four"
```

This ensures matching works regardless of capitalization.

---

# 🗑️ Removing the Book

Once a matching title is found:

```python
books.pop(i)
```

removes the book at index `i`.

Example:

Before deletion:

```python
[
    {"title": "Title One"},
    {"title": "Title Two"},
    {"title": "Title Three"}
]
```

If:

```python
books.pop(1)
```

is executed:

After deletion:

```python
[
    {"title": "Title One"},
    {"title": "Title Three"}
]
```

The element at index `1` is removed and the remaining elements shift left.

---

# 🚪 Why Use `break`?

After deleting the book:

```python
break
```

is executed.

```python
books.pop(i)
break
```

### Reason

Once the book is deleted:

- We already found what we were looking for.
    
- There is no need to continue searching.
    
- It avoids unnecessary iterations.
    

---

# 📌 Complete DELETE Endpoint

```python
@app.delete("/books/delete_book/{book_title}")
async def delete_book(book_title: str):

    for i in range(len(books)):

        if books[i].get("title").casefold() == \
           book_title.casefold():

            books.pop(i)
            break
```
