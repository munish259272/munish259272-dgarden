# First FastAPI App & Endpoint – Step-by-Step Tutorial

Let’s **build and run** your **first FastAPI application** from scratch in **PyCharm** using a **virtual environment**.

---

## Step 1: Open Project in PyCharm

1. Open **PyCharm**  
2. Click **Open** → Navigate to your `fastapi` project folder  
3. Select the folder → Click **Open**

> This loads your project with the virtual environment (`fastapi-env`) already created.

---

## Step 2: Set Up Python Interpreter (Critical!)

**Bottom-right corner**: Click **"No interpreter"** or current interpreter  
→ **Add New Interpreter** → **Add Local Interpreter**  
→ **Existing Environment**

### Navigate to your virtual env:
| OS | Path |
|----|------|
| **macOS/Linux** | `fastapi-env/bin/python` |
| **Windows** | `fastapi-env/Scripts/python.exe` |

Click **OK** → **OK**

> **Now PyCharm uses your isolated FastAPI environment**

---

## Step 3: Create `books.py`

Right-click project → **New** → **Python File**  
Name: `books.py`

---

## Step 4: Write Your First FastAPI App

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/api-endpoint")
async def first_api():
    return {"message": "Hello, Eric"}
```

### Code Breakdown

| Line | Purpose |
|------|--------|
| `from fastapi import FastAPI` | Import FastAPI framework |
| `app = FastAPI()` | Create your **API app instance** |
| `@app.get("/api-endpoint")` | **Decorator**: Maps `GET /api-endpoint` → function |
| `async def` | Optional but **recommended** (FastAPI auto-handles async) |
| `return {"message": ...}` | Auto-converted to **JSON response** |

> **Even `def` (without `async`) works** — FastAPI runs it in a thread pool.

---

## Step 5: Run with Uvicorn (Development Mode)

Open **Terminal** in PyCharm (bottom panel)

```bash
uvicorn books:app --reload
```

### Command Explained:
| Part | Meaning |
|------|--------|
| `books` | Your file: `books.py` |
| `app` | Variable where `FastAPI()` is assigned |
| `--reload` | **Auto-restart** on code save |

> Server starts at: **http://127.0.0.1:8000**

---

## Step 6: Test Your Endpoint

Go to browser:  
**http://127.0.0.1:8000/api-endpoint**

**Expected Output**:
```json
{"message": "Hello, Eric"}
```

---

## Bonus: Alternative Run Commands (New in FastAPI)

```bash
# Development mode
fastapi dev books.py

# Production mode
fastapi run books.py
```

> **We’ll use `uvicorn ... --reload` in this course** — more control & standard.

---

## Change Root Path (Optional)

Want response at root `/`?

```python
@app.get("/")
async def root():
    return {"message": "Hello, Eric"}
```

Now visit: **http://127.0.0.1:8000/**

---

## Stop the Server

In terminal:  
Press **Ctrl + C**

---

## Final `books.py` (Clean Version)

```python
app = FastAPI()

```
