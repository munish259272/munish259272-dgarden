## Understanding the Fields

|Attribute|Type|Purpose|
|---|---|---|
|`id`|`int`|Unique identifier|
|`title`|`str`|Book title|
|`author`|`str`|Author name|
|`description`|`str`|Description of the book|
|`rating`|`int`|Book rating|

---

# ⚙️ Step 7: Create a Constructor

The constructor initializes a book object whenever a new book is created.

```python
class Book:
    id: int
    title: str
    author: str
    description: str
    rating: int

    def __init__(
        self,
        id,
        title,
        author,
        description,
        rating
    ):
        self.id = id
        self.title = title
        self.author = author
        self.description = description
        self.rating = rating
```

---

## 📌 What Does `self` Mean?

> [!question]- What is `self` in a Python class?
> 
> `self` refers to the **current object instance** being created or used.
> 
> When Python creates a new object, it automatically passes that object as the first parameter.
> 
> Example:
> 
> ```python
> book1 = Book(
>     1,
>     "Python",
>     "Mukesh",
>     "Learning Python",
>     5
> )
> ```
> 
> Internally:
> 
> ```python
> Book.__init__(
>     book1,
>     1,
>     "Python",
>     "Mukesh",
>     "Learning Python",
>     5
> )
> ```
> 
> Here, `book1` becomes `self`.

# 🔄 What Happens When the Application Starts?

When FastAPI starts:

1. Creates the `Book` class
2. Executes the `books` list initialization
3. Creates six `Book` objects
4. Stores them in memory
5. Waits for incoming requests

```text
Application Start
        │
        ▼
 Create Book Class
        │
        ▼
 Create 6 Book Objects
        │
        ▼
 Store in books List
        │
        ▼
 Wait for Requests
```
