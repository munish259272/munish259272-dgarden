28.Class 55. Mar 17, 2024 : Does extracting date like this `df['InvoiceDate'] = pd.to_datetime(df['InvoiceDate']).dt.date` changes the datatype of the column `InvoiceDate` which was originally `datetime64[ns]` ? #flashcard 
Yes, when you use the `.dt.date` accessor with `pd.to_datetime()`, it will extract only the date component from the datetime objects and return a Series containing datetime.date objects. This effectively changes the data type of the column from `datetime64[ns]` to `object`, where each element is a Python `datetime.date` object.
To make sure you can perform datetime operations on it you can use astype to convert it to `datetime64[ns]` again as seen below:
```run-python

def calculate_iqr(data):
    sorted_data = sorted(data)
    n = len(sorted_data)
    Q1 = sorted_data[int(n * 0.25)]
    Q3 = sorted_data[int(n * 0.75)]
    IQR = Q3 - Q1
    return IQR
```
